Git & GitHub Practice

This repository is created to practice using Git and GitHub.

Objectives
Learn how to create and manage repositories
Practice commits with clear messages
Work with branches and pull requests
Features
Basic file modifications
Version control tracking
Branch creation and merging
Good Practices Applied
Small and descriptive commits
Use of branches for new features
Code review before merging
How to Use
Clone the repository
Create a new branch
Make changes and commit
Push to GitHub and open a pull request
Notes

This project is for learning purposes only and does not represent a production application.
